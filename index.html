SSH to SiteGround

首先在本機端創一個New key

ssh-keygen -t rsa
Enter file in which to save the key (/Users/.ssh/id_rsa): 這裡可以打檔案名稱(ir_rsa_test)若空白則是預設id_rsa)
再來按Enter即可

複製剛剛創好的public key
貼到Siteground -> DEVS -> SSH Keys Manager->Import

完成面步驟即可

接下來到電腦上使用ssh來測試連接

ssh -i ~/.ssh/id_rsa siteground使用者名稱＠網站domain -p18765

使用者名稱在siteground創好key的地方點擊Actions->SSH Credentials 就可以看到

Magento 介紹

Magento 是使用PHP為基礎所開發的電子商務平台(購物車系統)

擁有強大的各種訂單、客戶、金流管理功能、且有彈性，讓客戶或網站操作管理員都有很棒的服務。

由於在台灣 Magento 的技術文件不怎麼多，在開發研究多要往國外的文件去尋找，要熟練這個 framework 需要不少的時間，所以需要一些客製化的功能可以去尋找一些額外的Extension免費或付費 Magento Marketplace 或者網路上Opensource(Github 等等…) 讓你可以快速的找到你要的功能，節省自己開發的時間或做為開發的參考。
