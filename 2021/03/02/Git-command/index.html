Git command

Git Version 2.x

  • git 初始化目錄

      git init

    目錄底下會出現.git資料夾，目的是要讓 Git 開始對這個目錄進行版本控制。

  • 將檔案加入索引

      git add . `加入所有檔案的變化`
  git add -u `加入Modified & deleted 不包含new`

  • 加入版本資訊

      git commit

    可以在後面加入-m ，每一段的 message 都會被一個空行隔開 git commit -m text1 -m text2

  • 設定遠端節點(Github)

      git remote add origin git@github.com:yourname/testPractice-git.git

    add 指令是指要加入一個遠端的節點
    在這裡的 origin 指伺服器的位置 (這裡用Github為例)
    可以在資料夾 .git/config看到相關的設定

  • 設定好遠端節點，接下來可以把目錄的檔案推上去

      git push -u origin master

這裡是將local master branch 推到 origin上，如果remote 沒有master就會自動創建

  • 查看當前狀態

      git status

  • discard change

    git checkout –file

  • git clean清除未加入版控的資料
